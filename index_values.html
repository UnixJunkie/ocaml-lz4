C
compress [LZ4.S]
compress input returns LZ4-compressed input or raises Input_too_large if input is longer than 0x7E000000 bytes.
compress_bound [LZ4]
compress_bound sz returns the maximum size of the LZ4 output for sz bytes of input, or raises Input_too_large if sz > 0x7E000000.

D
decompress [LZ4.S]
decompress ~length input returns LZ4-decompressed input or raises Corrupted if input does not constitute a valid LZ4-compressed stream which uncompresses into length bytes or less.
