LZ4 compression algorithm
exception Input_too_large
exception Corrupted
val compress_bound : int -> int
compress_bound sz returns the maximum size of the LZ4 output for sz bytes of input, or raises Input_too_large if sz > 0x7E000000. Raises Invalid_argument "LZ4.compress_bound" if sz is negative.
